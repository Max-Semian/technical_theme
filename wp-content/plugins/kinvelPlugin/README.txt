Short codes:
[show_operator_card id='' sort='' sort_by=''] - отображает карточку оператора на главной странице
id - через запятую указываем id операторов (не указываем - получаем карточки всех видимых операторов)


[show_operator_page_card id='' sort='' sort_by=''] - отображает карточку оператора и карточку рейтинга на странице оператора
[show_bonus_page_card id='' sort='' sort_by=''] - отображает карточки бонусов на странице оператора

[show_rating_operator_cards id='' sort=''] - отображает рейтинг по указанному ID (id - обязательный параметр!)
В этом шорткоде сортировка производится по позициям рейтинга.

sort = 'ASC' - по возрастанию (default)
       'DESC' - по убыванию 
sort_by = ID — сортировка по ID.
          author — сортировка по ID авторов.
          title — сортировка по заголовку. (default)
          name — по названию поста (ярлык, слаг поста).
          date — сортировка по дате публикации.
          modified — сортировка по дате изменения.
          rand — случайный порядок.

Параметры не обязательны!

Functions:
get_operator ($id = 0, $sort = 'ASC', $sort_by = 'title') - получает всех видимых операторов

get_bonus ($id = 0, $for_operator=false, $sort = 'ASC', $sort_by = 'title') - получает все отображаемые на сайте бонусы
$for_operator=false - все операторы
$for_operator=true - отмеченные для отображения в карточке оператора

get_currency ($id = 0, $sort = 'ASC', $sort_by = 'title') - получает все записи валюты

get_rating ($id = 0, $sort = 'ASC', $sort_by = 'title') - получает все записи рейтинга 

get_geo ($id = 0, $sort = 'ASC', $sort_by = 'title') - получает все записи стран 

get_language ($id = 0, $sort = 'ASC', $sort_by = 'title') - получает все записи языков

Указанные типы постов получаются со всеми метаполями.

Параметры функций не обязательные
$id - ID желаемой записи (0 - все записи (default))

sort = 'ASC' - по возрастанию (default)
       'DESC' - по убыванию

sort_by = ID — сортировка по ID.
          author — сортировка по ID авторов.
          title — сортировка по заголовку. (default)
          name — по названию поста (ярлык, слаг поста).
          date — сортировка по дате публикации.
          modified — сортировка по дате изменения.
